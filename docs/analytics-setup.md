# Analytics Setup

## Goals

- Understand traffic quality.
- Measure product page conversion.
- Measure cart and checkout drop-off.
- Improve COD confirmation rates.
- Track AOV and upsell performance.

## Recommended Tools

- Meta Events Manager.
- TikTok Events Manager.
- Snapchat Ads Manager.
- Google Analytics 4 if desired.
- Backend order analytics from PostgreSQL.
- Google Sheets for operational confirmation.

## Core Metrics

- Product page view rate.
- Add-to-cart rate.
- Checkout open rate.
- Order submit rate.
- Upsell accept rate.
- COD confirmation rate.
- Duplicate/fake order rate.
- AOV.
- Revenue by product.
- Revenue by campaign.

## Funnel Events

- Product viewed.
- Offer selected.
- Add to cart.
- Cart opened.
- Checkout opened.
- Order submitted.
- Upsell shown.
- Upsell accepted.
- Thank-you viewed.

## UTM Discipline

Every paid ad should use:

- `utm_source`
- `utm_medium`
- `utm_campaign`
- `utm_content`
- `utm_term` when useful.

Example:

`?utm_source=tiktok&utm_medium=cpc&utm_campaign=chopper_launch&utm_content=video_01`
