# Colors and Typography

## Color Palette

Primary deep red:

- `#B91C1C`

Darker red:

- `#7F1D1D`

Black:

- `#111111`

White:

- `#FFFFFF`

Light gray background:

- `#F7F7F7`

Border gray:

- `#E5E7EB`

Text gray:

- `#4B5563`

Success green:

- `#15803D`

Warning amber:

- `#B45309`

## Usage Rules

- Red is for primary CTAs, selected offers, and important badges.
- Black is for headings and premium authority.
- Light gray is for background bands.
- Green is only for trust/success states.
- Amber is only for stock warnings or urgency.

Avoid a one-color page. The final look should be red, black, white, and gray with restrained accents.

## Typography

Use modern Arabic-friendly fonts.

Recommended:

- Arabic/UI: `Tajawal`, `Cairo`, or `IBM Plex Sans Arabic`.
- Latin fallback: `Inter`.

## Type Scale

Mobile:

- Hero heading: 30-36px.
- Section heading: 24-30px.
- Card heading: 17-20px.
- Body: 15-17px.
- Button: 15-17px.

Desktop:

- Hero heading: 48-64px.
- Section heading: 34-44px.
- Card heading: 20-24px.
- Body: 16-18px.

## Arabic Readability

- Use comfortable line-height: 1.7 for Arabic paragraphs.
- Avoid negative letter spacing.
- Do not scale font size with viewport width.
- Keep button copy short.
