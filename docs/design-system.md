# Design System

## Design Feel

Premium Moroccan minimal ecommerce:

- Clean layouts.
- High readability.
- Confident spacing.
- Red as brand accent, not overwhelming background.
- Black for authority.
- White and light gray for clarity.

## UI Shape

- Cards: max 8px border radius unless using existing shadcn defaults that are close.
- Buttons: strong rectangular shape with slight radius.
- Product offer cards: crisp borders and clear selected state.
- Avoid decorative blobs, orbs, and excessive gradients.

## Components

Use shadcn/ui as the base for:

- Button.
- Dialog.
- Drawer or Sheet.
- Input.
- Select.
- Accordion.
- Badge.
- Tabs where needed.
- Toast.

Use lucide-react icons:

- ShoppingCart.
- Menu.
- Star.
- ShieldCheck.
- Truck.
- Phone.
- CheckCircle.
- PackageCheck.
- X.
- Plus.
- Minus.

## Animation

Use Framer Motion sparingly:

- Drawer open/close.
- Offer selection.
- Section fade-in.
- Sticky CTA appearance.
- Upsell popup.

Animations should feel smooth, not playful or distracting.

## Section Rhythm

- Hero.
- Trust strip.
- Alternating content sections.
- Product grids.
- Review blocks.
- FAQ accordion.
- Final CTA.

## Accessibility

- Buttons must have clear labels.
- Dialogs must trap focus.
- Inputs must have labels.
- Error states must be readable.
- Color contrast must pass common accessibility checks.
