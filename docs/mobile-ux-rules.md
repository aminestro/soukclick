# Mobile UX Rules

## Priority

Souk Click is mobile-first. Most traffic will come from TikTok, Facebook, Instagram, and WhatsApp on mobile devices.

## Mobile Layout Rules

- First viewport must explain product value quickly.
- Sticky CTA must be visible on product pages.
- Avoid large empty hero sections.
- Keep text blocks short.
- Use large tap targets.
- Use product videos/images early.
- Keep checkout fields large and easy to complete.
- Avoid horizontal overflow.
- Avoid hidden CTAs below long content.

## Sticky CTA

Product pages must show sticky mobile CTA after user scrolls past hero:

- Product price or selected offer.
- Button: طلبي دابا
- Small COD reassurance.

## Header

Mobile header:

- Sticky top behavior.
- Logo on right side.
- Cart icon visible.
- Menu opens as drawer.

## Product Gallery

- Swiper enabled.
- First media should communicate the product instantly.
- Reels-style vertical cards for UGC section.
- Lazy load below-the-fold images.

## Checkout Popup

- Full-screen or bottom sheet on small phones.
- One column.
- Auto-focus first invalid field on error.
- Keep CTA sticky at bottom when keyboard is open if feasible.

## Typography

- Avoid tiny gray text.
- Body text must remain readable in Arabic.
- Buttons must not wrap awkwardly.
