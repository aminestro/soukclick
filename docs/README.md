# Souk Click Documentation

Souk Click is a premium Moroccan DTC ecommerce brand for smart home gadgets, organization products, practical daily solutions, and useful household products.

This docs folder is the operating manual for future developers and AI coders. Treat it as the source of truth before writing code.

## Core Goals

- High mobile conversion rate from TikTok and Facebook traffic.
- High cash-on-delivery confirmation rate.
- High average order value through bundles and checkout upsells.
- Strong Moroccan trust signals and premium brand presentation.
- Scalable product system for future household products.
- Clean engineering foundation using Next.js, FastAPI, PostgreSQL, Docker, and analytics-ready tracking.

## Required Stack

- Frontend: Next.js, React, TypeScript, TailwindCSS, shadcn/ui, Framer Motion, Zustand, TanStack Query, Zod, React Hook Form, SwiperJS.
- Backend: Python FastAPI, PostgreSQL, SQLAlchemy, Alembic, optional Redis, Docker.
- Deployment: Docker Compose and EasyPanel-ready services.
