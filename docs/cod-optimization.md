# COD Optimization

## Why COD Needs Special UX

Cash on delivery customers submit orders before payment. The store must reduce fake orders while making real customers feel safe.

## COD Trust Reassurance

Repeat near purchase actions:

- الأداء عند الاستلام.
- كنأكدو معاك الطلب قبل الإرسال.
- معلوماتك محمية.
- التوصيل داخل المغرب.

## Confirmation Rate Improvements

- Use Moroccan phone validation.
- Explain that a confirmation call or WhatsApp message will happen.
- Ask for complete address but keep the form short.
- Show order summary on thank-you page.
- Send clean order data to the confirmation team.
- Flag suspicious orders before fulfillment.

## Checkout Friction

Use only:

- Full name.
- Phone.
- City.
- Address.

Do not require account creation.

## Confirmation Team Fields

Google Sheets must include:

- Order ID.
- Created date.
- Customer name.
- Phone.
- Normalized phone.
- City.
- Address.
- Products.
- Quantities.
- Offer selected.
- Upsell status.
- Total MAD.
- Source UTM.
- Pixel IDs if available.
- Fraud status.
- IP country.
- Notes.
- Confirmation status.
